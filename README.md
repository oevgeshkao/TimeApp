# Мобильное приложение для обратного отсчёта

### 1\. Краткое описание проекта

Приложение позволяет пользователю выбрать дату начала и дату окончания события, после чего показывает:

* сколько дней осталось до конца;
* сколько прошло от начала;
* процент выполнения таймера;
* визуальный индикатор прогресса.

### 2\. Основные функции

* Ввод даты начала.
* Ввод даты конца.
* Расчёт:

&nbsp;	оставшихся дней,

&nbsp;	прошедших дней,

&nbsp;	общего времени,

&nbsp;	процента прогресса.

* Экран результата с прогресс-баром.
* Сохранение последнего события в локальное хранилище
* Простая навигация между экранами.



### 3\. Структура экранов

Экран 1 — Главный



Элементы:

* Заголовок: "Обратный отсчёт"
* Поля:

&nbsp;	Выбор даты начала

&nbsp;	Выбор даты окончания

* Кнопка "Рассчитать"



Экран 2 — Результат



Элементы:

* Заголовок: "До события осталось:"
* Крупным: X дней
* Подписи:

&nbsp;	Прошло: Y дней

&nbsp;	Всего: Z дней

* Кнопка: "Назад"



4\. Навигация



Главный экран → Результат

(после ввода дат и нажатия «Рассчитать»)



Результат → Главный

(нажатие «Назад»)
